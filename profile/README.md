## Maximum とは？

Maximum は、国際大学対抗プログラミングコンテスト (ICPC) へ参加し、良い成績を上げるために成立したプログラミング研究会です。
2022 年度からは、競技プログラミングのみならず、 Web やその他のプログラミングの知識を高める活動を行っています。
